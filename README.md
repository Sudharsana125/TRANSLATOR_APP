# Translation App

A simple web application for text translation using MyMemory API.

## Features
- Translate text between multiple languages
- Auto-detect source language
- Copy translations to clipboard
- Text-to-speech for translations
- Character count
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Node.js
- Express
- MyMemory API

## Installation
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open http://localhost:3000

## Usage
1. Enter text in the left box
2. Select source and target languages
3. Click "Translate" or press Ctrl+Enter
4. Use copy button or speaker icon for additional features

## API
This app uses the (Gemini API)
